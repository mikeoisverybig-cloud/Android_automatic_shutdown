# Android_automatic_shutdown
A Magisk module with automatic shutdown

有多个备用机或者需要上交手机的学生党，都对不小心把手机开机导致没电感到烦恼 所以我写了这个模块

## ⚠️ 免责声明（必读）
> 本模块为**个人技术实验项目**，非商业产品，不保证兼容所有设备/ROM。  
> - **核心逻辑**：充电状态检测 > 通知滑除取消 > 安全关机。**插入充电器是 100% 保底取消方式**，通知交互受 OEM 后台策略限制。  
> - **不承诺**：全机型兼容、后台唤醒稳定性、通知 100% 不丢。  
> - **欢迎反馈**：提交 Issue 时请附 `adb logcat | grep shutdown_timer` 和 `dumpsys battery` 输出，**仅处理有日志的反馈**。  
> - **无义务**：不提供一对一技术支持，不接受「为什么我手机没反应」类提问。
